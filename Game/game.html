Regras:

  • Cada célula pode estar em um de três estados: oculta, revelada ou marcada.
  • Para revelar uma célula, clique nela.
  • Para colocar uma bandeira, clique com o botão esquerdo.
  • Se uma célula revelada contiver um número, ele indica quantas minas estão adjacentes a essa célula.
  • Se uma célula não revelada for clicada e conter uma mina, o jogo termina e você perde.
  • Se todas as células sem minas forem reveladas, você vence.
